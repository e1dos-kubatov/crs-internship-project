# crs-internship-project

## Run (2 terminals)
1. Backend:
```powershell
cd C:\Users\Admin\crs-internship-project\car-rental-backend
mvn spring-boot:run
```
2. Frontend:
```powershell
cd C:\Users\Admin\crs-internship-project\car-rental-system\frontend
npm install
npm run dev
```

Frontend: `http://localhost:5173`  
Backend: `http://localhost:8081`

```powershell
cd C:\Users\Admin\crs-internship-project
.\run-dev.ps1
```
